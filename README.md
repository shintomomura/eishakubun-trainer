# 瞬間英作文トレーナー

日本語の文を見てとっさに英語で言い換える「瞬間英作文」をブラウザでランダム練習できるシンプルなWebアプリです。

## 主な機能

- **ランダム出題** — 全420問をシャッフルして出題
- **自己採点** — 「答えを見る」で正解を表示し、`○ できた / × できなかった` を自分で記録
- **復習モード** — 「できなかった」とマークした問題だけを再出題（できるまでループ）
- **進捗の自動保存** — 中断してもブラウザを閉じても、`できなかったリスト` と現在の進行状況は LocalStorage に保存され続けます
- **キーボードショートカット**
  - `Space` / `Enter` … 答えを見る
  - `1` または `O` … ○ できた
  - `2` または `X` … × できなかった
  - `S` … スキップ（あとで戻ってくる）
  - `Ctrl/⌘ + Enter`（入力欄内）… 答えを見る → ○ で進む
- **モバイル対応** — スマホでもそのまま使えるレスポンシブUI、ライト/ダーク自動切替

## 使い方

1. このリポジトリの GitHub Pages URL（オーナーが有効化済みであれば）にブラウザでアクセスする
2. 日本語の文が表示されるので、頭の中で（または入力欄に）英作文してみる
3. 「答えを見る」を押して正解を確認
4. 自分の出来に応じて `○ / ×` をタップ
5. 全問終わったら、「復習モード」で「できなかった」だけを再挑戦

## ローカルで動かす

`fetch('problems.json')` を使うため、ファイルを直接 `index.html` で開くと CORS エラーになります。簡易HTTPサーバーを立ててください。

```sh
# Python 3
python -m http.server 8080
# その後 http://localhost:8080 を開く
```

## 問題データのソース

問題はGoogleスプレッドシート（**閲覧公開**設定）から起動時に直接取り込んでいます。

- スプレッドシートに行を**追加・編集・削除**するだけで、次回ブラウザを開いたときに自動で反映されます
- スプレッドシートにアクセスできない場合（オフライン・公開停止等）は、リポジトリに同梱されている `problems.json` のスナップショットに自動フォールバック
- 各問題のIDは `ja + en` の内容ハッシュ（FNV-1a 32bit）。行を入れ替え／削除しても、内容が同じ問題は同じIDを保つので「× できなかった」リストの整合性が壊れません
- 削除された行はLocalStorageの「できなかった」リストからも自動的に除外されます（次回起動時のクリーンアップ）

データ形式（フォールバック用 `problems.json`）:
```json
[
  { "id": "h_xxxxxx", "ja": "日本語の文", "en": "English answer" },
  ...
]
```
